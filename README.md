流媒体合租也就是发车，是指由陌生的网友自行组织，一起合购某个流媒体平台的家庭账号，其中的发起人就是车主，而流媒体账号的=允许的子账号数量会决定此次发车的车位数量。比如Netflix家庭账号允许建立五个子账号，每个子账号可以设置不同的密码（PIN码），所以奈飞发车一般会有五个车位，允许5位用户一起合购。
